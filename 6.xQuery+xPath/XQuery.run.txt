Requiring XQuery Environment:

#1 use IDE itself: Altova XMLSpy

#2 use BaseX, which requires Java 11 or higher (i.e.: JDK 19)
    - Install Java 11 or higher version first 
      Source:  https://www.oracle.com/java/technologies/downloads/#jdk19-windows

    - Install BaseX (default user: admin, password: define yourself, i.e.: Admin000)
      Source: https://basex.org/download/

    - Open "BaseX GUI" from start menu

#3 or Use command line XQuery tool: 
    i.e.: https://www.videlibri.de/xidel.html
